# 🏋️ AI Fitness Trainer

A real-time AI-powered fitness trainer that tracks your exercises using computer vision and counts repetitions automatically.

---

## 🚀 Features

* 🎥 Real-time pose detection using MediaPipe
* 🔢 Automatic rep counting (Squats & Push-ups)
* 🌐 Web-based interface using Streamlit
* 📱 Clean dark UI (mobile-app inspired)
* 🧠 Works directly in browser using WebRTC

---

## 🧠 Tech Stack

* Python
* OpenCV
* MediaPipe
* Streamlit
* Streamlit-WebRTC

## ⚙️ Installation (Local Setup)

1. Clone the repository:

```bash
git clone https://github.com/your-username/AI-Fitness-Trainer.git
cd AI-Fitness-Trainer
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```

4. Open browser and allow camera access 🎥

---

## 🌐 Deployment

This app is deployed using **Render**.

⚠️ Important:

* WebRTC is used for camera access
* STUN server configuration is required for deployment

---

## 📌 Notes

* Uses MediaPipe Pose for body tracking
* Rep counting is based on joint angle detection
* Best results with **side view posture**
